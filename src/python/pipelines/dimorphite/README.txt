This package uses dimorphite-dl from the Durrant lab at University of Pittsburg.
For details see:
Ropp PJ, Kaminsky JC, Yablonski S, Durrant JD (2019) Dimorphite-DL: An open-source
program for enumerating the ionization states of drug-like small molecules.
J Cheminform 11:14. doi:10.1186/s13321-019-0336-9.

The original Dimorphite-dl code can be found here:
https://git.durrantlab.pitt.edu/jdurrant/dimorphite_dl
The two files dimorphite_dl.py and site_substructures.smarts are copied here.
